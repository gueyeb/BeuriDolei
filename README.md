# BeuriDolei

Application iOS de défi planche (plank challenge) — SwiftUI.

## Stack
- iOS 17+ · SwiftUI · Swift 6
- MVVM · UserDefaults pour la persistance
- Bundle ID : `com.dakhine.BeuriDolei`

## Lancer le projet
Ouvrir `BeuriDolei/BeuriDolei.xcodeproj` dans Xcode.

## Structure
```
BeuriDolei/
└── BeuriDolei/
    ├── BeuriDoleiApp.swift   # entry point
    ├── ContentView.swift     # vue principale
    └── Assets.xcassets/
```
